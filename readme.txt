API digunakan untuk mengcrawling data dari website orami.co.id dan bukalapak.com berdasarkan kategori produknya, 
kemudian menampilkan data hasil crawling dalam bentuk html dan menampilkan produk dengan harga tertinggi dan terendah.
API ini dibuat oleh 
Nadya Anastasia / 18217020
Lidya Jessica / 18217037

Web API dapat dijalankan dengan mengetik:
localhost:8000/<nama kategori>

List nama kategori produk yang bisa dijalankan pada web API gabungan dari crawling data website orami.co.id dan data website bukalapak.com:
<nama> (<orami> | <bukalapak>)

Kereta bayi (kereta-bayi | keretabayi)
Perlengkapan mandi (keperluan-mandi-bayi | perlengkapanmandi)
Mainan (seni-dan-kreatifitas | mainan)
Handuk (handuk | handuk)
Makanan bayi (makanan-bayi | makananbayi)
